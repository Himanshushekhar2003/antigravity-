# Local Delivery App — Product Blueprint

## Overview
A hyperlocal delivery app covering a 10 km range, built on a zone-based model with batch delivery, smart rider routing, dynamic pricing, and a robust complaint/fraud system.

## Core Systems
1. **Zone-Based Delivery & Minimum Orders**
2. **Batch Delivery & Rider Route Optimization**
3. **Real-Time Matching System**
4. **Wrong/Damaged Food Handling System (Complaint API)**
5. **Fraud Detection System**

## Tech Stack
- Frontend: Flutter (UI Screens)
- Backend: Node.js / Firebase
- Optimization Logic: Python
- Database: Relational schema defined (orders, complaints, penalties)
